# Olimpiādes sagatavošanas programma (ātri, bet pakāpeniski)

Šī ir **īsa ceļa** mācību programma skolēnam, lai nonāktu līdz:
- sarakstiem, 2D sarakstiem (matricām),
- stekam/rindai,
- linked list,
- kokiem un grafiem,
- algoritmu šabloniem: two pointers, sliding window,
- DP pamatiem.

## Kā mācīties (strādājošs režīms)
Katram tematam:
1) izlasi pamatu,
2) pārraksti šablonu kodu,
3) atrisini uzdevumus (vismaz 2–3),
4) paskaties NeetCode risinājumu piemērus (video/risinājums) un **centies izpildīt uzdevumu pēc atmiņas**.

## Failu struktūra
- **01–12**: temati secībā
- **99**: īsā špikerlapa
- **13**: mini-kontroldarbi + ieteiktais grafiks
