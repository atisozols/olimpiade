# Olimpiādes sagatavošanas programma (ātri, bet pakāpeniski)

Šī ir **īsa ceļa** mācību programma skolēnam, kurš jau zina **ciklus un sazarojumus**, lai nonāktu līdz:
- sarakstiem, 2D sarakstiem (matricām),
- stekam/rindai,
- linked list,
- kokiem un grafiem,
- algoritmu šabloniem: two pointers, sliding window,
- DP pamatiem.

## Kā mācīties (strādājošs režīms)
Katram tematam:
1) izlasi pamatu,
2) pārraksti šablonu kodu,
3) atrisini uzdevumus (vismaz 2–3),
4) paskaties NeetCode risinājumu piemērus (video/risinājums) un **pārraksti no atmiņas**.

> Ja iestrēgst > 20 min: uzraksti ievadi/izvadi → bruteforce → tikai tad skaties ideju.

## Failu struktūra
- **01–12**: temati secībā
- **99**: īsā špikerlapa
- **13**: mini-kontroldarbi + ieteiktais grafiks
